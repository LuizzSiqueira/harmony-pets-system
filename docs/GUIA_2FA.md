# Guia de Autenticação de Dois Fatores (2FA) - Harmony Pets

## O que é 2FA?

A Autenticação de Dois Fatores (2FA) adiciona uma camada extra de segurança à sua conta. Além da sua senha, você precisará de um código gerado pelo Microsoft Authenticator no seu telefone.

## Como Configurar

### 1. Pré-requisitos
- Instale o **Microsoft Authenticator** no seu smartphone
- Faça login na sua conta do Harmony Pets

### 2. Ativando o 2FA

1. Acesse seu **Perfil** no menu
2. Na seção "Configurações de Segurança", clique em **"Configurar"**
3. Abra o Microsoft Authenticator no seu telefone
4. Toque em **"+"** para adicionar uma conta
5. Escolha **"Conta corporativa ou de estudante"** ou **"Outra conta"**
6. Escaneie o QR Code exibido na tela
7. Digite o código de 6 dígitos que aparece no app
8. **IMPORTANTE**: Guarde os códigos de backup em local seguro
9. Clique em **"Ativar 2FA"**

### 3. Como Usar

Após ativar o 2FA:

1. Faça login normalmente com seu usuário e senha
2. Você será redirecionado para a tela de verificação
3. Abra o Microsoft Authenticator
4. Digite o código de 6 dígitos exibido para o Harmony Pets
5. Clique em **"Verificar Código"**

### 4. Códigos de Backup

Os códigos de backup são essenciais caso você:
- Perca seu telefone
- Desinstale o app por engano
- Troque de telefone

**Cada código só pode ser usado uma vez!**

Para usar um código de backup:
1. Na tela de verificação, clique em **"Use um código de backup"**
2. Digite um dos códigos de 16 caracteres
3. O código será invalidado após o uso

### 5. Desativando o 2FA

⚠️ **Atenção**: Desativar o 2FA torna sua conta menos segura.

1. Acesse seu **Perfil**
2. Na seção de segurança, clique em **"Desativar"**
3. Digite sua senha atual
4. Digite o código atual do Microsoft Authenticator
5. Confirme que entende os riscos
6. Clique em **"Desativar 2FA"**

## Segurança

### Boas Práticas:
- ✅ Mantenha os códigos de backup em local seguro
- ✅ Use uma senha forte e única
- ✅ Mantenha o Microsoft Authenticator atualizado
- ✅ Configure backup da conta Microsoft

### O que NÃO fazer:
- ❌ Não compartilhe seus códigos com ninguém
- ❌ Não tire screenshot dos códigos de backup
- ❌ Não desative o 2FA sem motivo importante

## Solução de Problemas

### Código não funciona
- Verifique se o horário do telefone está correto
- Certifique-se de digitar o código rapidamente (expira em 30s)
- Tente sincronizar o horário no Microsoft Authenticator

### Perdi o telefone
- Use um código de backup
- Entre em contato com o suporte para desativar o 2FA

### App não gera códigos
- Verifique a conexão com internet
- Reconfigure a conta no Microsoft Authenticator
- Use um código de backup temporariamente

## Compatibilidade

O sistema é compatível com:
- ✅ Microsoft Authenticator (recomendado)
- ✅ Google Authenticator
- ✅ Authy
- ✅ Qualquer app TOTP padrão
